# Noteworthy（词弦）项目策划

## 一、名字释义

### 英文名：Noteworthy

“Noteworthy”是一个巧妙的双关词，核心在于 **note** 的双重含义：

- **note = 笔记 / 单词**：指向语言学习模块，记录值得记忆的词汇。
- **note = 音符 / 乐音**：指向音乐学习模块，聆听值得辨认的和弦。

后缀 **-worthy** 意为"值得的"，整体构成两层语义：
- **值得记住的单词**——每一次打开都是一次词汇积累。
- **值得聆听的音符**——每一组和弦都是一次听觉训练。

单词发音干净利落，字形优雅，天然适合作为应用名称，兼具书籍质感和音乐气质。

### 中文名：词弦

“词弦”以极简的两字组合，直接概括两个模块的核心：

- **词**：单词、语言、词汇。代表语言学习模块。
- **弦**：和弦、弦乐、音乐。代表音乐练耳模块。

两字并列，一文一武，暗示两个看似不同的领域在学习底层上的统一。简洁有力，适合中文界面或副标题使用。

---

## 二、项目定位

**Noteworthy（词弦）** 是一款双模块的轻量级学习工具，核心理念是：**利用碎片时间，通过偶然性输入激发主动学习。**

软件不追求系统的课程体系或进度压力，而是将学习拆解为每一次打开时的"瞬间练习"——像闪卡一样不经意地出现在你面前，让学习变成一种自然而然的日常仪式。

两个模块看似方向不同（视觉 vs 听觉、文字 vs 音乐），但底层共享同一套学习逻辑：

> **偶然输入 → 主动辨认**

每一次随机出现的内容，都是一次有意识的认知练习。

---

## 三、模块设计

### 模块一：语言学习（词 / Word）

**核心体验**：每次打开软件，随机展示一个单词。

以闪卡式交互呈现，附带以下辅助信息，帮助用户从多个维度理解并记忆单词：

- **释义**：准确的中英文释义
- **例句**：真实语境中的用法
- **词根**：词源拆解，辅助理解构词逻辑
- **搭配**：常见词组与固定搭配

用户在不经意间积累词汇，无需刻意背诵，依靠重复曝光和上下文线索自然内化。

### 模块二：音乐学习（弦 / Chord）

**核心体验**：软件随机生成一套和弦或单个和弦，播放音频供用户聆听，用户需要辨认和弦的性质、级数或色彩。

以听觉反馈为核心机制，训练内容包括：

- **和弦性质**：大三、小三、增三、减三、属七、大七、小七等
- **和弦级数**：在特定调性中判断 I、IV、V、vi 等
- **和弦色彩**：辨识紧张/松弛、明亮/阴暗等听感特征

通过反复的"聆听 → 辨认 → 验证"，逐步培养对和弦的直觉感知能力。

## 四、设计原则

| 原则 | 说明 |
|------|------|
| **轻量** | 无复杂课程体系，无进度压力，打开即用，用完即走 |
| **随机** | 每次遇见的内容不可预测，保持新鲜感和好奇心的驱动力 |
| **碎片** | 适配任意空闲时长，哪怕只有 30 秒也能完成一次练习 |
| **主动** | 随机只是入口，辨认和思考才是核心——用户始终需要主动参与 |
| **统一** | 两个模块形式不同，但交互节奏和学习逻辑保持一致 |

## 五、目标用户

- 想利用碎片时间学习，但不希望有课程负担的人
- 对语言和音乐都感兴趣，希望一个工具覆盖两种需求的人
- 认同"偶然学习"理念，愿意将学习融入日常习惯的人

## 六、技术栈

### 框架：Flutter

**选型理由**：一套 Dart 代码同时覆盖 Windows 和 Android，两端官方支持成熟；卡片式 UI 与 Flutter Widget 模型天然契合；热重载提高开发效率。

### 音频：MIDI 合成

- **和弦生成**：纯 Dart 层构造 MIDI 消息（note on/off、velocity、program change），两平台共享同一套生成逻辑。
- **Windows 播放**：Platform Channel 调用系统 Microsoft GS Wavetable Synth（`winmm.dll` 或 `Windows.Devices.Midi`）。
- **Android 播放**：通过 `flutter_midi` 或 Android MIDI API 发送 MIDI 指令播放。

### 词汇数据

- 本地 JSON/SQLite 词库，包含释义、例句、词根、搭配等字段。
- 支持后续扩展为在线词库或导入自定义词表。

### 界面

- 极简卡片式设计，以内容为核心，减少操作层级。

## 八、如何运行

### 环境要求

- Flutter SDK ^3.11.5
- Android SDK（Android 构建）
- Visual Studio 2022 含桌面 C++（Windows 构建）
- Windows 开发者模式（Windows 原生插件构建需要）

### 运行命令

```bash
# 安装依赖
flutter pub get

# Web（两个模块都可交互，MIDI 无声音）
flutter run -d chrome

# Windows（完整 MIDI 合成支持）
flutter run -d windows

# Android（完整 MIDI 合成支持）
flutter run -d <android-device-id>
```
